# Fixture Mundial FIFA 2026

Aplicación web para consultar el fixture del Mundial FIFA 2026, resultados, tablas de posiciones, mejores terceros y fase eliminatoria.

## Tecnologías

- React
- Node.js
- Express
- PostgreSQL

## Funcionalidades

- Registro e inicio de sesión de usuarios
- Fixture por fecha
- Resultados y partidos pendientes
- Tablas de posiciones
- Mejores terceros
- Fase eliminatoria
- Panel de administración
- Gestión de usuarios
- Sincronización automática de partidos
- Horarios adaptados a la zona horaria local del usuario
